# EcoSense_AI
An AI-powered conversational assistant promoting sustainable living through the Sustainable Development Goals (SDGs).

🌱 EcoSense AI – Sustainability Assistant

EcoSense AI is an AI-powered sustainability assistant built using Botpress Cloud to promote awareness about Sustainable Development Goal (SDG) 13 – Climate Action. The chatbot engages users through interactive conversations and provides practical suggestions to encourage environmentally responsible habits and sustainable living.

This project was developed as part of the Lenovo LEAP NextGen Scholar Program, where I explored conversational AI, prompt engineering, workflow design, and chatbot development through a real-world implementation.
